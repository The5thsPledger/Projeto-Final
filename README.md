# Como Iniciar
```sh
# no diretorio raiz:
npm i # instala dependências

npm run start:dev # com --watch
# ou
npm run start # sem --watch
```

## Como Testar

```bash
# no diretorio raiz:
$ npm test # testar todos os módulos
$ npm teste [marca | usuario | veiculo] # testar módulo especificado
```
